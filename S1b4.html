Đây là trang web cảu tôi

Chào mừng bạn đến với trang web của tôi. Đây là 1 ví dụ về cách sử dụng thẻ img để hiển thị hình ảnh trên 1 trang web

Trên đây là 1 hình ảnh ngẫu nhiên được sử dụng từ trang web unsplash bạn có thể thay đổi hình ảnh bằng cách chỉnh độ rộng chiều cao trong thuộc tính "width" và "heigth của thẻ. bạn có thể thêm mô tả hình ảnh bằng thuộc tính "alt."
